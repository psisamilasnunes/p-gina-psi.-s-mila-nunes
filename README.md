# 🌐 Sâmila Nunes | Psicóloga Clínica

Este projeto consiste em uma página web institucional desenvolvida para apresentar os serviços da psicóloga **Sâmila Nunes**, com foco em atendimento clínico, identidade visual acolhedora e navegação simples.

---

## 📌 Sobre o projeto

A página foi construída utilizando **HTML e CSS puro**, com o objetivo de oferecer:

* Apresentação profissional
* Identidade visual elegante e acolhedora
* Experiência de navegação fluida
* Layout responsivo (adaptado para dispositivos móveis)

---

## 🎨 Design e Estilo
